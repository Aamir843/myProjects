Steps to initialize and Run this Assignment :

1. Install the Packages using following commands:
   a. npm install
   b. npm install express
   c. npm install nodemon  
   d. npm install mssql
   e. npm install msnodesqlv8
   f. npm install mysql
   g. npm install handlebars
   h. npm install bootstrap
   

2. Update the Server name in connection.js file 

3. Run the command - "node connection.js".   

4. Create a new database named "ResultManagement" using Sql server studio.
   And create a table named "Results" having columns "Name" , "RollNumber" , "DateOfBirth" , "Score".

5. Save and run the command "npm start" and go to "localhost:3306" .
